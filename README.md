# Medical Insurance Cost Prediction

## Overview
This project analyzes medical insurance data to identify factors affecting insurance charges and predict high-cost insurance cases using machine learning techniques in R.

## Dataset
Dataset: Medical Insurance Cost Prediction  
Source: Kaggle  
https://www.kaggle.com/datasets/rahulvyasm/medical-insurance-cost-prediction

Place the dataset file `medical_insurance.csv` in the project directory before running the script.

## Machine Learning Models
The following models were implemented and compared:
- Logistic Regression
- Linear Discriminant Analysis (LDA)
- K-Nearest Neighbors (KNN)

## Key Analyses
- Age distribution analysis
- BMI analysis
- Smoking impact on insurance charges
- Regional insurance trends
- Correlation matrix visualization
- High-charge classification

---

## How to Run
1. Download the dataset from Kaggle.
2. Place `medical_insurance.csv` in the project folder.
3. Open the R script in RStudio.
4. Run the script.
